# 最直观的 Typecho 主题开发框架  

> The Intuitive Theme Develop Framework for Typecho.

配置要求: PHP 8.1+ Typecho 1.2+

## 框架特点  

 - 轻量
 - TyAjax
 - REST API
 - 钩子系统
 - 路由系统
 - 选项框架
 - 快速调用
 - 字段查询
 - 数据库查询
